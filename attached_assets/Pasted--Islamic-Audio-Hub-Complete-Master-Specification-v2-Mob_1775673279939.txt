1. PROJECT OVERVIEW
App Name: Islamic Audio Hub
Tagline: செவிகள் சிறக்கட்டும்! சிந்தனை மாறட்டும்! (Let ears flourish! Let thoughts change!)
Purpose: A Tamil Islamic audio learning platform — Quran Tafsir, Hadith explanations, Seerah, and daily guidance in Tamil, presented as an audio streaming app with gamified quiz system.
Target Audience: Tamil-speaking Muslims
Language: UI is primarily in Tamil, with some English labels
Theme: Dark theme by default (charcoal + gold/amber accent), with light theme (Islamic green) toggle
Published URL: https://iah.lovable.app

2. DATABASE SCHEMA
2.1 categories
CREATE TABLE categories (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  type TEXT NOT NULL DEFAULT 'general',
  sort_order INTEGER NOT NULL DEFAULT 0,
  is_enabled BOOLEAN NOT NULL DEFAULT true,
  thumbnail_url TEXT,
  audio_url TEXT,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

2.2 subcategories
CREATE TABLE subcategories (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  category_id UUID NOT NULL REFERENCES categories(id),
  name TEXT NOT NULL,
  description TEXT,
  icon TEXT DEFAULT '📖',
  sort_order INTEGER NOT NULL DEFAULT 0,
  sort_order_start INTEGER NOT NULL,
  sort_order_end INTEGER NOT NULL,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);
Subcategories map tracks by sort_order range.

2.3 tracks
CREATE TABLE tracks (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  category_id UUID REFERENCES categories(id),
  title TEXT NOT NULL,
  audio_url TEXT NOT NULL,
  thumbnail_url TEXT,
  template_id INTEGER DEFAULT floor(random()*4+1)::int,
  view_count INTEGER NOT NULL DEFAULT 0,
  is_premium BOOLEAN NOT NULL DEFAULT false,
  is_daily_free BOOLEAN NOT NULL DEFAULT false,
  free_date DATE,
  duration INTEGER DEFAULT 0,
  sort_order INTEGER NOT NULL DEFAULT 0,
  transcript TEXT,
  prize_enabled BOOLEAN NOT NULL DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);
2.4 track_segments
CREATE TABLE track_segments (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  track_id UUID NOT NULL REFERENCES tracks(id),
  audio_url TEXT NOT NULL,
  segment_order INTEGER NOT NULL DEFAULT 0,
  duration INTEGER DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);
2.5 quiz_questions
CREATE TABLE quiz_questions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  track_id UUID NOT NULL REFERENCES tracks(id),
  question_number INTEGER NOT NULL,
  question_text TEXT NOT NULL,
  option_a TEXT NOT NULL,
  option_b TEXT NOT NULL,
  option_c TEXT NOT NULL,
  correct_option TEXT NOT NULL,   -- 'அ', 'ஆ', or 'இ'
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);
2.6 quiz_rewards
CREATE TABLE quiz_rewards (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL,
  track_id UUID NOT NULL REFERENCES tracks(id),
  prize_category TEXT NOT NULL,
  prize_group TEXT NOT NULL,      -- 'MEN', 'WOMEN', 'KIDS'
  token TEXT NOT NULL UNIQUE,
  score INTEGER NOT NULL DEFAULT 0,
  total_questions INTEGER NOT NULL DEFAULT 0,
  time_taken_seconds INTEGER NOT NULL DEFAULT 0,
  is_winner BOOLEAN NOT NULL DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now()
);
2.7 user_profiles
CREATE TABLE user_profiles (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL UNIQUE,
  is_premium BOOLEAN NOT NULL DEFAULT false,
  premium_until TIMESTAMPTZ,
  subscription_status TEXT NOT NULL DEFAULT 'free',
  subscription_expiry_date TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);
2.8 user_roles
CREATE TYPE app_role AS ENUM ('admin', 'user');
CREATE TABLE user_roles (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL,
  role app_role NOT NULL DEFAULT 'user',
  created_at TIMESTAMPTZ DEFAULT now(),
  UNIQUE(user_id, role)
);
2.9 favorites
CREATE TABLE favorites (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL,
  track_id UUID NOT NULL REFERENCES tracks(id),
  created_at TIMESTAMPTZ DEFAULT now()
);
2.10 playback_progress
CREATE TABLE playback_progress (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL,
  track_id UUID NOT NULL REFERENCES tracks(id),
  progress_seconds NUMERIC NOT NULL DEFAULT 0,
  updated_at TIMESTAMPTZ DEFAULT now(),
  UNIQUE(user_id, track_id)

2.11 audio_notes
CREATE TABLE audio_notes (

2.13 comments
CREATE TABLE comments (

2.14 payment_orders (Razorpay)
CREATE TABLE payment_orders (

4. AUDIO PLAYER SYSTEM
4.1 Core Features
Global persistent player across all pages
Collapsed view: Thumbnail, title, play/pause, speed, share, quiz icon
Expanded view (mobile): Full controls with seek bar, speed selector (0.5x–2x pills), prev/next, volume
Default speed: 1.25x
Background playback continues when tab is hidden
Visibility handling resumes on tab return
4.2 Queue & Autoplay
Queue management (add/remove, visual badge)
Playlist autoplay from category tracks
Multi-segment sequential playback
4.3 Playback Progress
Saves every 10 seconds (DB if logged in, localStorage otherwise)
5. PREMIUM/MONETIZATION SYSTEM
5.1 Access Logic
FREE if: track.is_premium === false OR track.is_daily_free === true
UNLOCKED if: Gold Member OR purchased track
LOCKED if: is_premium AND not Gold AND not purchased
5.2 Payment (Razorpay)
Single track: ₹29 | Gold subscription: ₹99/month
QR code payment option for UPI
Auto-play after successful payment with unlock animation

6. QUIZ GAME SYSTEM 🎯
6.1 Entry Points
Track Detail Page: Large yellow button
Audio Player (mini): Quiz icon with yellow pulse dot
Track Cards: "QUIZ" badge + quiz icon
GameLevelCard: Primary "Play Quiz" button
Direct URL: /quiz/:trackId
6.2 Quiz UI (Full-Screen Game Modal)
Gradient background (charcoal to card) with glow orbs
Header: Track title + Level badge + Global timer + Close button (red glow, X icon)
Close behavior:
Click outside modal → closes
Click X button → closes
Press ESC → closes
Returns to same page, preserves scroll position
Answer buttons, audio player controls are "safe zones" (don't trigger close)
6.3 Quiz-First Learning Flow
User attempts quiz first
"🎧 விளக்கம் கேளுங்கள்" (Audio Hint) button available
When audio plays → quiz timer auto-pauses (both per-question and global)
When audio stops → quiz auto-resumes
Paused state shows overlay with headphones icon
6.4 Question Screen
Progress: "3 / 10" badge + gradient progress bar
Streak counter: 🔥 streak at 2+ consecutive correct
Score: ⭐ points display
Per-question timer: 12 seconds countdown
Global timer: 60 seconds for entire quiz
Question card: Gradient background with bold Tamil text
3 options: Tamil letters அ, ஆ, இ
Correct: Green glow + bounce
Wrong: Red + shake, correct shown
Points popup: "+1" or "+2" floats up
Auto-advance: 1.5s after answer
6.5 Scoring
Correct: +1 | Streak bonus (3+): +2
Time up: 0 points, streak resets
6.6 Prize/Reward System (Admin-Controlled)
Admin toggle: prize_enabled field on tracks (ON/OFF)
IF ON: Pre-quiz prize category selection screen:
MEN: Casual Items, Accessories, Gadgets, Premium Gifts
WOMEN: Beauty Items, Accessories, Personal Care, Premium Gifts
KIDS: Toys, Creative Kits, Chocolates & Treats, Premium Gifts
Complete quiz within time → generate unique token (e.g., MEN-A3X9K)
Token stored in quiz_rewards table
Winner screen shows token with copy button
IF OFF: Normal quiz without prizes
6.7 Result Screen
Trophy with pulse animation
Score: "X / 10" with star rating (0-3 stars)
Performance message in Tamil
Buttons: Play Again, Next Level, Close
6.8 Level Progression
Each track = 1 Level
Auto-fetches next track's quiz in same category
"All levels completed" message when done
